#### Welcome to the TodoRoo App

TodoRoo is a Todo List App built for the purpose of CADDi online test in a span of 3 hours.

The app uses TypeScript, ReactJS and can be used to add and remove todos.

In the project directory, you can run:

##### `npm install` to install dependencies

##### `npm start` to start the development server on port 8000

Open [http://localhost:8000](http://localhost:8000) to view it in the browser.

##### `npm run test` to run the unit tests
